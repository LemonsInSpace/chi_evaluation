# Masked Keccak Chi Leakage Study

This repository contains the Chi-focused firmware and analysis scripts used for the experiments in the accompanying paper.

The original experimental firmware contained additional Keccak components and test code. For clarity, this repository has been reduced to the parts relevant to the reported Chi experiments:
- masked Chi implementation
- intentionally broken Chi variant
- capture harness
- analysis scripts

It is intended as an experimental artifact for the reported leakage study, not as a complete Keccak implementation.
